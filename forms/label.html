dando utilidade aos formulários

uma coisa simples e necessária em todos os formulários é, o local para onde os dados vão, normamelmente esses dados são mandados para alguma linguagem de programação, por enquanto utilizaremos PHP

para enviarmos os dados usaremos o "action" na tag >form< e isso vai ditar a ação que vai ser disparada ao clicarmos no botão de submit

usando label

explicando a diferença entre name e id

você deve ter notado que ao usar o input:text é criado um name e um id.

para a linguagem php funcionar direito, é necessário que todo input tenha um name, então esse nome é bem mais util para o html e o php, linguagens como o javascript o id é bem mais importante

finalemente utilizando label

ao envelopar o "nome" na tag >label< ele terá apenas um valor o valor 'for=""'.

dentro desse for vamos botar o id

e assim conseguimos relacional o label ao input que queremos, visualmente isso não muda nada, mas esse é um passo EXTREMAMENTEimportante para a criação de formulários

a utlizidade do label é relacionar uma palavra a um objeto, isso facilita o uso nos celulares, e a acessibilidade nos sites
