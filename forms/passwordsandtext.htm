Validando um formulário

apesar de termos criado um formulário melhorado, ele ainda tem vários erros, como por exemplo, o formulário é enviado mesmo se o campo estiver vazio

para impedirmos isso de acontecer teremos que usar o atributo required

<input type="text" name="*" id="*" required >

esse atributo vai tornar obrigatório que o campo seja preenchido antes de ser validado

esse atributo pode ser ainda mais reforçado, por exemplo, se você quiser botar um minimo e um máximo de letras é possivel usar o minlength="" ou maxlength="" depois do required

<input type="text" name="*" id="*" required minlength="5" maxlength="15">

legibilidade também é importante para formulários, nesse tipo de caixa, podemos usar um placeholder para botar um texto de explicação ou um exemplo para guiar o usuário

<input type="text" name="*" id="*" required minlength="5" maxlength="15" placeholder="Digite seu nome">

as vezes o tamanho da caixa pode atrapalhar na acessibilidade de dispositivos com telas menores, para isso podemos utilizar o atributo size

o size determinao tamanho da janela, ele especifica quantas letras podem aparecer ao mesmo tempo

<input type="text" name="*" id="*" required minlength="5" maxlength="15" placeholder="Digite seu nome" size="5">
