Inputs

continuando a falar sobre os diferentes inputs dessa vez vamos falar do input para email

e-mail

muitos formulários precisam de um email

esse formulário exige que o email seja validado pelo sistema

essa validação por hora é bem simples, ela checa apenas se o email tem um @ na composição

Tel

da mesma forma que o email, a validação desse campo não é muito extensa inicialmente

para criar um campo que tem uma validação melhor é necessario estudar sobre, expressões regulares ou regEx

agrupando os inputs

existe uma tag que é usada para agrupar grupos de elementos, isso torna a legibilidade bem melhor, o nome dessa tag é >fieldset<
