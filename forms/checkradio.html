como você pode ter notado, só da pra selecionar uma opção se você clicar diretamente na caixa, o que não é um grande problema se tiver usando um computador pessoal, mas caso você use um celular esse se torna um problema maior

radio

outro input bastante popular com uma função similar ao checkbox é o radio, também tem a funcionalidade de marcação do checkbox, mas dessa vez é limitado a apenas uma escolha

vamos criar uma e dessa vez vamos pré-aplicar o label

Sexo

como você pode ver dessa forma, você ainda pode marcar os 2 ao mesmo tempo, e não só isso, você também não pode desmarcar nenhum dos 2

por que isso acontece?

isso acontece porque o elemento name="#" está diferente nas duas opções de radio

            <input type="radio" name="sexmas" id="isexmax"><label for="isexmax">Masculino
                    input type="radio" name="sexfem" id="isexfem"><label for="isexfem">Feminino

para o input radio funcionar direto é extremamente importante que as alternativas inclusas nesse rádio tenham o mesmo nome, dessa forma, se mudarmos o nome de ambas as opções para ficar igual teremos o seguinte resultado

Sexo

também é possivel deixar uma opção pré marcada se usarmos o elemento checked DENTRO do input desejado

outra coisa importantissima para nosso formulário, é botar um valor nos botões radio, caso contrario vai ser enviado um value de sexo="on", par resolver isso é só coloar um value com o retorno necessário

checkbox não vai ter esse problema já que ele só envia os valores selecionados
