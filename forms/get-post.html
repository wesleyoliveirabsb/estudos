No HTML existem 2 metodos simples para envio de formulário

Método GET

GET é o método padrão usado pelo HTML quando não se define nenhum método

Método POST

É o método alternativo de envio de formulário, as vezes considerado um pouco mais seguro do que o GET

Usar GET ou POST?

É recomendado usar o GET quando os dados requisitados não forem sensiveis como por exemplo nome, peso ou qualquer coisa nesse aspecto.
Geralmente alguma pesquisa ou caso você queira que o usuário possa voltar para essa mesma página exata depois(ou compartilhar essa págica com alguma outra pessoa)

GET também tem um limite de dados que podem ser enviados de uma só vez, então se você precisa mandar alguma coisa que tenha mais de 3 mil bytes, é necessário usar o POST

Já o método POST é usado para formulários que precisam de mais seguranças onde dados podem ser utilizados para modificar um banco de dados, ou alguma página que você não quer que deixem marcada

também é recomendado usar o POST para envio de arquivos
