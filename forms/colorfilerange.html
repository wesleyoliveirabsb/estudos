Color

menos usado mas não menos importante é o input de cor, é um input que permite que o usuário escolha uma cor

se você quiser adicionar algum value para o input color é necessario usar o valor hexadecimal por exemplo: #00ff00

esse tipo vai retornar um valor em hexadecimal para o formulário

o texto está depois da caixa para lembrar que é importante organizar o html, caso o label estivesse antes do input estária no lugar certo

range

range é um input mais raro mas também usado para saber coisas como, o nivel de satisfação das pessoas em pesquisas

como podemos ver o range tem o formato de um slider

ele retorna um valor de 0 a 100 ao formulário por padrão mas isso pode ser modificado usando o min e max

file

esse é bastante popular e com certeza você já viu por ai em algum site

é extremamente importante usar o method="post" para esse tipo de controle (envio de arquivos), lembre-se use post sempre que o formulário tiver mais do que 3000bytes
