Formas de proteger o seu site

como dito anteriormente outros sites podem usar o seu site para capturar informações do usuário

Vamos discutir como proteger um pouco melhor o seu site

sandbox

o sandbox bloqueia todo e qualquer acesso que outro site dentro iframe tentem fazer, como por exemplo cadastrar alguém no formulário acima

referrerpolicy

esse atributo vem com o valor padrão de "no-referer-when-downgrade", mas alterando isso para "no-referrer"

usando o no-referrer, você diz que o conteudo que está carregado dentro do iframe, não vai coletar nenhum tipo de informação do usuário

altamente recomendado

mas e se eu precisar de preencher o formulário em um site?

existem vários valores para o sandbox que vai permitir as coisas, o sandbox="sandbox" cria a segurança máxima, mas você pode substituir esse valor sandbox por exemplo sandbox="allow-forms" que permite o uso de formulários, ou sandbox="allow-scripts" que permite o uso de scripts.

mais de uma valor pode ser adicionado ao mesmo sandbox

lembre-se que o sandbox vai permitir apenas o que você especificar, todas as outras coisas ainda estarão bloqueadas
