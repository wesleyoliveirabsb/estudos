Problemas com o iframe

apesar do iframe ser muito útil existem uma série de problemas que torna o uso dele difícil

problemas com mecanismos de busca

como por exemplo o google bot tem dificuldades de anexar o conteudo presente no iframe, então é recomendado não adicionar conteudos essenciais ou sensiveis dentro do iframe

problemas com acessibilidade

pessoas com algum tipo de necessidade especial, como por exemplo, pessoas cegas, elas conseguem usar sim computadores utilizando leitores de tela, esses softwares leitores de tela, também apresentam alguns problemas para conseguir acessar os iframes

problemas de usabilidade

são aqueles que o proprio navegador, ou até mesmo o usuário acaba se confundindo

por exemplo, alguns navegadores não conseguem decidir se o botão de voltar, está se referindo a pagina dentro do iframe ou fora do iframe

sites que usam target="_blank" ou seja os que tentar abrir uma janela nova, alguns navegadores abrem essa janela dentro do iframe e não como uma aba nova na tela principal navegador

Problemas com responsividade

Se o site dentro do iframe não for responsivo, pode ser que o site apareça todo quebrado dentro do iframe, a culpa é de quem fez o outro site por não torna=lo responsivo, mas ainda é problema seu, por você querer usar um site não-responsivo

Problemas com segurança

O iframe abre uma porta para outros sites, e é muito importante ter cuidado com essas portas, se existe um site dentro do iframe que foi invadido e está usando tecnicas de captura de informações do usuário, o seu site acaba fazendo isso em conjunto com o site invadido, aumentando o alcance de usuários afetados

essa segunraça é responsabilidade sua, preste muita atenção nos sites que você vai colocar no iframe

tipos mais comuns de falha de segurança dos iframes

Ataques cross site ou XSS

Os ataques XSS → permitem que os invasores injetem scripts do lado do cliente em páginas da Web visualizadas por outros usuários. Uma vulnerabilidade de script entre sites pode ser usada por invasores para contornar controles de acesso, como a política de mesma origem.
