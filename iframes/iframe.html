O QUE É IFRAME?

Iframe significa frame inline

utilizando o iframe

acesse meu onde se encontram os projetos

o iframe cria uma nova instancia de navegador dentro de um site já aberto

alguns sites possuem uma proteção contra iframes, como por exemplo o proprio google

hoje em dia a maioria dos sites usa essa proteção, mas não impede que a gente aprenda como usar

mas alguns sites ainda permitem o uso livremente

Configuração do iframe

por padrão o iframe tem 300x150 px, mas isso pode ser mudado facilmente alterando os valores dentro do proprio iframe, além de obviamente, ser possivel alterar tudo com CSS

scrolling

scrolling é o parametro de configuração da barra de rolagem de um iframe, pode ter os valores auto(o padrão), yes(vai forçar uma barra de rolagem), e no(que tira a rolagem por completo)

frameborder

é um parâmetro que vai dizer se existe ou não uma borda no iframe, aceita os valores, 0 e 1, sem borda e com borda respectivamente
