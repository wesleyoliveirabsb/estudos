O que é media query?

se você já criou qualquer site em html, você pode ter notado que as vezes o site tem problema para se adaptar a um tamanho diferente de tela, fica ou muito grande ou muito pequeno.

media query surgiu para dar um jeito nesse problema, dessa forma você pode permitir que seu site se adapte a diferentes condições, e até mesmo quando for imprimir uma coisa do seu site
