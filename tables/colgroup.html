agrupando colunas

é fácil agrupar linhas e aplicar o css desejado apenas selecionando o tr, mas como selecionamos colunas inteiras?

colgroup

para agruparmos uma coluna utilizaremos o colgroup, uma tag nova adicionada no html5 utilizada para agruparmos colunas.

para isso vamos adicionar um <colgroup> e um <col> para<cada coluna que existe na tabela como no exemplo a seguir

            <colgroup>
                <col>
                <col>
                <col>
                <col>
            </colgroup>

essa tag deve ser colocada dentro da tag <table> dessa forma cada <col> dentro dela representa uma coluna, assim podemos facilmente aplicar class e id para cada uma delas

também podemos adicionar um <span> para indicar que a tag representa duas colunas ou mais

        <colgroup>
                <col>
                <col>
                <col span="2">
            </colgroup>
