uso de tabelas grandes

tabelas grandes, por recomendação da w3c, precisam de uma estrutura diferente e mais semântica do que tabelas pequenas

a hierarquia de uma tabela pequena como visto anteriormente é simples 

        table
            table row ou tr 
                table header 
                table data

mas ao trabalhar com tabelas grandes precisamos mudar um pouco a hierarquia sendo assim:


        table
            caption = é o titulo da tabela
            thead 
                tr, td, th 
            tbody 
                tr, td, th 
            tfoot 
                tr, td, th

âmbos caption, e o tfoot, podem ser colocados em qualquer lugar da hierarquia mas no produto final elees sempre vão estar no lugar
